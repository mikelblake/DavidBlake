Mission

My life’s work and personal mission is to help create an educational system that will:
1. Drive the cost of learning to zero 2. Promote universal access to education 3. Create meaningful indicators of personal educational outcomes and success 4. Help organize meaningful curricula and establish interoperable standards 5. Restructure the physical learning environment 6. Enable a measured and meaningful system for lifelong learning
